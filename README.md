# Verse Music

**Verse-Music** is a music-bot that supports  
a lot of sources, including _Youtube_ and  
_Spotify_, it also have a playlist system  

<img src='./img/bot.png' width='300px'>

## Installation

 - Download [Node][node] and [Java][java] (You probably already have it)
 - Download and unzip this repository 
 <img src='./img/download.png' width='600'>
 
 - Complete `config.json`
 - Execute `install.bat`
 - Now the bot is ready, when you want to use it,  
run `bot.bat`
  
A list with all commands can be found [here][commands]
